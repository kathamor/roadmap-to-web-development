How to insert images using HTML

  • The HTML Image Element

    The Image tag is a self-closing tag, so it doesn't need to be closed, but for the image element just providing an image tag isn't enough you also need to specify the source, and that's the soruce of the image that will be displayed, so that can be either url so it can fetch it from the internet, or it can be a local image that's included in the same directory as your website.


  • The src Attribute

    The required src attribute specifies the path (URL) to the image. Note: When a web page loads; it is the browser, at that moment, that gets the image from a web server and inserts it into the page. Therefore, make sure that the image actually stays in the same spot in relation to the web page, otherwise your visitors will get a broken link icon. The broken link icon and the alt text are shown if the browser cannot find the image.
